Step 1: Download the installer
  - Go to: https://cloud.google.com/sdk/docs/install
  - Click "Windows" (if not already selected)
  - Click "Installer (.exe)"
  - Download google-cloud-sdk-*.exe

Step 2: Run the installer
  - Double-click the .exe file
  - Follow the installation wizard
  - Accept all defaults
  - It will ask to install Python - let it

Step 3: Restart your computer
  - Close and reopen PowerShell

Step 4: Verify installation
  - Open PowerShell
  - Run: gcloud --version
  - Should show version number

Step 5: Authenticate
  - Run: gcloud auth login
  - Browser opens
  - Log in with your Google account
  - Click "Allow"

Step 6: Set your project
  - Run: gcloud config set project YOUR_PROJECT_ID
  - Replace YOUR_PROJECT_ID with your GCP project

Now you can use gcloud commands!
