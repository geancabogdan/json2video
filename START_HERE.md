## ğŸ¯ Common Tasks

### Upload a Video File

1. Select a clip
2. Choose layer type: "Video File"
3. Click the file input
4. Select a .mp4, .mov, or .avi file
5. Automatic upload

### Change Video Quality

| Use Case | Width | Height | FPS | Notes |
|----------|-------|--------|-----|-------|
| Preview/Test | 640 | 360 | 15 | Fast |
| Standard | 1280 | 720 | 30 | Good quality |
| HD | 1920 | 1080 | 30 | High quality |
| 4K | 3840 | 2160 | 24 | Very slow |

### Add Text Over Video

1. Add Clip with video
2. Switch layer type to "Title"
3. Enter text
4. Render

Both video and text render together!

### Multiple Clips with Transitions

1. Add Clip 1 (video)
2. Set transition: "Cross Fade", duration: 1s
3. Add Clip 2 (image)
4. Set transition: "Slide Left", duration: 1s
5. Add Clip 3 (color)
6. Render

Clips transition automatically!

## ğŸ’» API Quick Reference

If you want to use the API instead of the UI:

### Upload File
```bash
curl -F "file=@video.mp4" http://localhost:5000/api/upload
```

### Render Video
```bash
curl -X POST http://localhost:5000/api/render \
  -H "Content-Type: application/json" \
  -d '{
    "clips": [{
      "duration": 4,
      "layers": [{"type": "fill-color", "color": "#667eea"}],
      "transition": {"name": "crossFade", "duration": 0.5}
    }],
    "width": 1280,
    "height": 720,
    "fps": 30,
    "outPath": "./outputs/test.mp4"
  }'
```

### Check Status
```bash
curl http://localhost:5000/api/render/{jobId}/status
```

## ğŸ”§ Troubleshooting

### "Cannot find module 'canvas.node'" (Windows)
```bash

```

### Port 5000 already in use
```bash
PORT=3000 npm run dev
```

### FFmpeg not found
- Download: https://ffmpeg.org/download.html
- Or install:
  - Mac: `brew install ffmpeg`
  - Ubuntu: `sudo apt-get install ffmpeg`

### Video renders slowly
- Reduce resolution (640Ã—360)
- Reduce FPS (15 fps)
- Use shorter clips
- Close other apps

### Upload fails
- File size must be < 500MB
- Format must be supported (mp4, jpg, png, etc)
- Check disk space
